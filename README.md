# Build and Battle

This is an HTML5 card and dice game playable in the browser.

## How to deploy on GitHub Pages

1. Create a new GitHub repository and push all files from this folder.
2. Go to repository settings > Pages.
3. Set the source to the `main` branch and `/root` (or `/docs` if you move files).
4. Your site will be available at `https://<your-username>.github.io/<repo-name>/`.
